== Description ==

Easy & accurate ECommerce sales tax rate data. Use Taxjar integration to simplify your Sales Tax collection.

== Installation ==

1. Install the Taxjar - WooCommerce Plugin 
2. Activate the plugin
3. Visit taxjar.com/api to get your API key
4. Go to the Taxjar Settings Page 
5. Enter your API key and business address
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use
